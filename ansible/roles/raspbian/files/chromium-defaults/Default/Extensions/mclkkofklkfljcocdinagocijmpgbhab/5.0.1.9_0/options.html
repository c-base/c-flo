Synchronize user dictionary:   This is for IMEs. By selecting this, the words you have entered and the associated source strings will be saved and synchronized on Google’s server as your personal dictionary. It will be used to improve the conversion accuracy. If you uncheck this, we will stop saving your input on Google’s server. Note that your personal dictionary that has been stored on Google’s server will not be cleared if you uncheck this option.

Show on-screen keyboard:   If you select a virtual keyboard, you can show or hide the on-screen keyboard by this option. But you can input native script without looking the keyboard layout.

Show status bar:   Status bar is the small floating window used by Chinese Pinyin, Wubi, Cangjie, Zhuyin, and Cantonese IMEs to switch Chinese/English mode, Chinese/English punctuations, and full-wdith/half-width alphanumeric. You will see it when you have selected one of these IMEs.
